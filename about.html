  • librg: redefined! Jan 2020 - now
    This moment has shaped librg into a powerful library with a simple purpose: Entity data streaming. Narrowing the scope down to entity streaming, we've optimized our library for use in any environment desired. librg would allow us to pack streamable data within a grid-based world without dependency on networking setup or user application.
  • Bye C++, hello C99! Jul 2017
    We've decided to reboot ReGuider Framework under a new name: librg. This rewrite would drop support for scripting and resource management in favor of more flexible and powerful entity data streaming. It relied on a networking library called ENet which would connect players together and stream entity data on a grid-based world. We've used this iteration of a library on a project called Mafia Oakwood which was an unofficial multi-player modification for a legendary game.
  • We've founded the zpl! 2017 - now
    A fork of gingerBill's gb library, zpl has shaped up to provide a wide range of additions, improvements and design changes that would improve C/C++ development. Over time, we've implemented a support for various platforms and architectures. The library went through many changes until it became the ❤ of our organisation.
  • Our legacy Jan 2017
    The first version of our library, called librg, would surface under ReGuider Framework. Originally designed to provide a suite of tools used for game dev multiplayer development, it has handled a wide range of tasks, from creating a fully networked game session, scripting (Squirrel), and resource management to network-bind culling. We've used it internally to develop multiplayer solutions for our in-house projects.
